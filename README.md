# Encom OS MiniDisk Interface

A browser-based **MiniDisc interface and recording planner** inspired by classic hardware aesthetics (ENCOM / TRON, Matrix, Walkman, macOS Sequoia).

This tool helps you:
- Plan MiniDisc recordings (SP / LP2 / LP4)
- Check if tracks **will fit** before recording
- Account for **real-world disc time already used**
- Simulate gaps between tracks
- Visualize disc usage with charts
- Design printable MiniDisc labels
- Enjoy multiple retro-futuristic themes
